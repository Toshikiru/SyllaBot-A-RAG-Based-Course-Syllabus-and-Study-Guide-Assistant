1. course description
systems analysis and design (sad) provides students with a structured approach to designing and
developing information systems. the course covers the systems development life cycle (sdlc),
requirements gathering, process modeling using data flow diagrams (dfds) and use case diagrams,
database design, prototyping, and system implementation. students will work in teams to analyze a
real-world problem and produce a complete system design proposal.
2. learning outcomes
lo 1: apply the sdlc methodology to plan and manage a software development project.
lo 2: elicit and document system requirements through interviews, surveys, and observation.
lo 3: create process models using dfds, flowcharts, and uml diagrams.
lo 4: design a functional prototype of an information system based on user requirements.
